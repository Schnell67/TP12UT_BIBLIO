Java pour les nuls

Java pour les nuls (Barry Burd, 2019)

Idéal pour commencer avec Java comme premier langage ! Grâce à ce livre, vous allez rapidement écrire rapidement vos premières applets Java, sans pour autant devenir un gourou de la programmation objet. Rassurez-vous, on ne vous assommera pas avec toutes les subtilités du langage Java, mais vous posséderez rapidement les bases nécessaires pour utiliser la panoplie d'outils du parfait programmeur Java. Cette nouvelle édition a été entirement mise à jour avec les spécifications de la dernière version du SDK Java. Intègre les nouveautés apportées par Java 8.
